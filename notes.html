World Wide Web


Aims:

-A basic understanding of how the internet works
-Understand how Users, computers, servers, HTTP and the Internet fit together.

-The World Wide Web A Collection of HTML Documents Created in the 1990s and comprises of around 30 billion pages. The documents can contain videos, pictures, music and plain text files.

-HTML (HyperText Markup Language) This is what the web runs on and glues everything together.

-URL (Uniform Resource Locator) the www. address typed in the browser.

-HTTP (Hypertext Transfer Protocol) HTTP is a Protocol that handles your computers request from the server via the Internet. Server then sends back files which your browser displays.

-Server A computer that hosts files and sends them to your computer via the HTTP protocol.

In a nutshell

Cient(computer) talks to server via the internet using http on a browser.
